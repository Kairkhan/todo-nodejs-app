# todo-nodejs-app


A barebones Node.js app using [Express 4](http://expressjs.com/).

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
$ git clone https://github.com/Kairkhan/todo-nodejs-app.git # or clone your own fork
$ cd todo-nodejs-app
$ npm install
$ npm start
```
